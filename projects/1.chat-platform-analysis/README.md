# 🚀 聊天平台用户行为分析

## 项目来源与致谢

本项目是基于 **黑马程序员** 大数据教程的学习实践作品。

- **学习视频**：[B站：黑马程序员 - Hive实战教程](https://www.bilibili.com/video/BV1WY4y197g7)
- **参考案例**：课程中的聊天数据分析
- **个人实践内容**：
  - 环境搭建与配置（Hadoop + Hive + FineBI）
  - FineBI 可视化仪表板制作

## 📊 项目简介

本项目基于某聊天平台的30万条消息数据，使用 **Hive** 进行数据清洗和统计分析，并通过 **FineBI** 制作可视化仪表板，旨在洞察用户活跃规律、设备偏好和地域分布。

## 🎯 项目目标

- 掌握 Hive 数据清洗和 ETL 流程
- 熟悉 HiveSQL 数据统计分析
- 用 FineBI 制作可视化仪表板

## 📁 数据集

- **数据来源**：模拟聊天记录数据集 `chat_data-30W.csv`，来源于黑马程序员教学案例。
- **数据量**：30万条记录，19个字段

## 🛠️ 技术栈

| 工具 | 用途 |
|------|------|
| Hadoop | 分布式存储与计算 |
| Hive | 数据清洗与统计分析 |
| DBeaver | SQL 开发与调试 |
| FineBI | 数据可视化 |
| Git | 版本控制 |

## 📋 分析问题

实现了以下8个分析问题：

1. 统计每日消息总量
2. 统计每小时消息量趋势
3. 统计各地区发送消息总量（基于GPS）
4. 统计每日发送/接收用户人数
5. 统计发送消息最多的 Top 10 用户
6. 统计接收消息最多的 Top 10 用户
7. 统计发送人的手机型号分布
8. 统计发送人的操作系统分布
